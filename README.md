# SmartEdu
## Introduction
A Smart Education system allow you to:
1. Load documents and get them analyzed autonomously
2. Convert resources into knowledge graph
3. Have support from a Virtual Teahing Assistancce that:
    - Get videos, documents or any learning resources suitable for you
    - Get information and answer question that related to the scalable enormous source of Knowledge
    - Give learning recommendation, tracking your performance
